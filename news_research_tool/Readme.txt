Project: Equity market Research::
Research analyst will use this tool to get summary answer of different-different articles related to the stocks.


why not use chatGPT?
1. Copying and pasting articles in ChatGPT is a tedious task.
2. We need aggregate knowledge base answer (Chatgpt doesn't know what was the last quarter sell and may be answers are in different articles)
3. chatgpt word limit
    GPT-3.5:
        Token Limit: 4,096 tokens
    GPT-4:
        8k context model:
             Token Limit: 8,192 tokens
        32k context model:
               Token Limit: 32,768 tokens


Embedding: allow identifying/figerout a relevant chunk
Vector database: can perform faster search on the database
